Places links to donate towards Typhoon Haiyan Relief Efforts on your page, post or theme via widget and/or shortcode. 

== Installation ==
1. Upload 'typhoon-haiyan-donate-links' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the widget to place the donation links on your sites sidebar. 
1. Use the shortcode [typhoon_help] to place the donate links on a post or page.
